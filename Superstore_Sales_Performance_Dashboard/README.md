# Superstore Sales Performance 

## Project Overview
This project analyzes retail sales data using Microsoft Excel to identify trends,
top-performing regions, products, and categories.

## Tools Used
- Excel Formulas (IF, SUMIF, COUNTIF)
- Pivot Tables
- Charts and Slicers

## Excel Sheets Included
- Raw_Data (original dataset)
- Helper (derived columns)
- Sanity_Check (data validation)
- Condition_Logic (business logic checks)
- Pivot_Analysis (summary tables)
- Final_Dashboard (interactive dashboard)

## Key Insights
- Sales are highest during year-end months
- West region drives the most revenue
- Technology is the top-performing category
- A small percentage of high-value orders contributes significantly to total sales

## Outcome
Built an interactive Excel dashboard to support data-driven business decisions.
